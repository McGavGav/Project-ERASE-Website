## Work Summary
In this sprint the team focused on creating the primary functions desired by the client. We implemented the user authorization/authentication system for admin and super user roles to allow for board member specific actions on the site. We implemented the calendar system for event posting to allow for RSVPing and information communication. We got a student database prototype up and running and a map prototype implemented. On the developer side we cleaned up code to allow for easier page implementation through base extension.
